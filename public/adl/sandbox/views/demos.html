Multi-User Demo
In this recorded demonstration, you'll see several users interacting to find and apprehend a suspect in a desert environment. The scene is set up for night, and each player has a light. The helicopter is equiped with a large searchlight that is tied to the direction of the thermal camera. The demo involves 4 users- one to fly, one manning the camera, and 2 on the ground. The suspect is an NPC controlled by simple scripts
Communication
While this video does not show it exactly, we have run this demo with 4 RTC connections to allow each user to communicate. In this recording, everyone was in the same room.
Interactivity
One of the great parts of this exercise is that it really works. The handheld lights are really not enough to search the area, and the suspect is quite hard to find without good communication from the thermal camera. Likewise, the pilot really cannot see the people below, and must work with the camera operator to aim the spot light.
